# Wiki application

### This web application is used to read and add interesting articles

To run local:
1. Clone repository
```bash
git clone https://github.com/00014548wiut/CWWT_00014548.git
```
2. Install dependencies
```bash
cd [project folder]
npm install
```
3. Run the app
```bash
node app.js
```

### Dependepncies
- express.js
- pug.js
